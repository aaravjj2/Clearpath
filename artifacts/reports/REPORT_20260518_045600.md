## Before / After

**IMP-001:**
- Before: `answer = answer_question(...)` → returned `<coroutine object answer_question>`, JSON serialization would fail
- After: `answer = await answer_question(...)` → returns the actual string answer

**IMP-002:**
- Before: No limit; 100 MB PDF would be fully read into memory before rejection
- After: HTTP 413 with friendly message if file > 20 MB or text > 500k chars

**IMP-003:**
- Before: SSE error events logged to browser console only; user saw a frozen progress bar
- After: Red alert banner with `role="alert"` appears in the UI when any SSE error occurs

**IMP-004:**
- Before: `collection.query(n_results=3)` on a 1-document collection raised `chromadb.errors.InvalidArgumentError`
- After: `safe_n = min(n_results, count)` prevents the error; empty collection returns `[]`
