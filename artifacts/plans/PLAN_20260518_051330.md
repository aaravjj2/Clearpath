- SSE streams through proxies (nginx, Vercel) can be killed after 30–60s idle;
  a keepalive comment ping every 15s prevents this.
- No "copy to clipboard" button on simplified clause text (power UX).
- `backend/tests/` directory doesn't exist; tests live at root of backend.
  Creating a proper `tests/` folder with `conftest.py` improves pytest discoverability.
- `ProviderRegistry.get_available_providers()` never calls `OllamaProvider.check_available()`
  so Ollama shows as "available" even when not running.
